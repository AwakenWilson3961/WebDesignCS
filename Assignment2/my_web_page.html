Block-level Elements:

  • Always starts on a new line
  • Takes up the full width available(streches out to the left and right as far as it can).
  • Has a top and a bottom margin, whereas an inline element does not
  • A few block-level elements: heading elements, p,ol, ul, hr and div

Inline-level Elements

  • Does NOT start on a new line
  • Only takes up as much width as necessary
  • A few Inline-level elements: b, em, a, img, br, sub, sup
